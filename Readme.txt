一、项目简介
本系统是一个基于JSP/Servlet的新闻信息平台，支持新闻发布、审核、分类展示及用户评论反馈等功能。
技术栈：
后端：Java 8、JSP/Servlet、MySQL
前端：Bootstrap、Layui、UEditor
服务器：Tomcat 8.x
数据库连接池：C3P0
二、运行环境要求
1. JDK 1.8+
2. Tomcat 8.x
3. MySQL 5.7+
4. 浏览器：Chrome/Firefox/Edge 等现代浏览器
三、项目配置
修改数据库连接信息：
编辑src/main/resources/c3p0-config.xml，将数据库的用户名以及密码改写为你的数据库用户名以及密码。
